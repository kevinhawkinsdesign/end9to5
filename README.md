# End9to5

A landing page for **End9to5** — a 6-month program for mid-career operators ready to leave their job and build something real.

## About

End9to5 combines operational frameworks from Kevin Hawkins (20+ years scaling teams) with transformational coaching from Ora Özer (15+ years in tech and film). The program is designed specifically for people who are still employed and building on the side.

## What's in this repo

- `index.html` — Single-page site with all content, styles, and scripts inline.

## Site sections

- Hero + CTA
- Company logos marquee
- Mission statement
- 6-module curriculum (Introduction → Decide → Clarity → Positioning → Ship → Scale)
- Benefits grid
- Competitor comparison table
- Kevin & Ora bio sections
- Case study video
- Testimonials carousel
- Pricing (€1,200 one-time)
- Email signup (Mailchimp-ready)
- FAQ accordion (19 questions)
- Footer

## Deployment

Deployed via [Netlify](https://end9to5.netlify.app/).

## Setup notes

- The Mailchimp form action URL in `index.html` needs to be updated with your account's form endpoint before the email signup will work.
- Logo images for company marquee reference external URLs — replace with local assets if needed.
